l# Brotherhood 🏆

> A daily discipline tracker for four friends — synced in real time via Firebase.

---

## Features

| Screen | What it does |
|---|---|
| **Home** | Today's date, greeting, completion %, remaining tasks, streak |
| **Calendar** | Daily dots — Green (all done) · Yellow (partial) · Red (mostly missed). Starts 12 July 2026 |
| **Members** | All four members' completion %, today's tasks, streak |
| **Leaderboard** | Ranked by Completion % and Streak |
| **Admin (Vansh only)** | Add / Edit / Delete tasks, change reminder times |

**Default Tasks:** Drink Water · Workout · Study · Read Book · Sleep Before 11 PM · No Smoking

## Setup (5 Steps)

### Step 1 — Install Flutter

Download Flutter SDK from https://flutter.dev/docs/get-started/install and add it to your PATH.

```bash
flutter doctor   # should show ✅ Flutter · ✅ Android toolchain
```

### Step 2 — Create Firebase Project

1. Go to https://console.firebase.google.com
2. Click **Add project** → name it `brotherhood` (or anything)
3. Enable **Cloud Firestore** (Build → Firestore Database → Create database → **Production mode**)
4. In Firestore **Rules** tab, paste the contents of `firestore.rules` and click **Publish**
5. In Project Settings → **Your apps** → Add app → Android
   - Package name: `com.brotherhood.app`
   - App nickname: Brotherhood
   - Click **Register app**
6. Download `google-services.json` → place it at `android/app/google-services.json` (replacing the placeholder)

### Step 3 — Configure FlutterFire

```bash
# Install FlutterFire CLI globally
dart pub global activate flutterfire_cli

# In the flutter-brotherhood directory:
cd flutter-brotherhood
flutterfire configure --project=<your-firebase-project-id>
```

This will overwrite `lib/firebase_options.dart` with your real credentials.

### Step 4 — Install Dependencies

```bash
flutter pub get
```

### Step 5 — Build the APK

```bash
# Debug APK (for testing)
flutter build apk --debug

# Release APK (for sharing)
flutter build apk --release
```

The APK will be at:
```
build/app/outputs/flutter-apk/app-release.apk
```

Share this APK with all four friends via WhatsApp / Google Drive / AirDrop.

## First Launch

On first launch each phone shows an **identity selection screen**.  
Each friend taps their name → it's saved permanently (SharedPreferences).  
No login, no password, ever.

> **Vansh** gets the Admin tab where he can add/edit/delete tasks and change reminder times.

---

## Notifications

Notifications are scheduled locally on each device using `flutter_local_notifications`.  
They fire daily at the configured times:

| Task | Default Time |
|---|---|
| Drink Water | 08:00 |
| Workout | 07:00 |
| Study | 10:00 |
| Sleep Before 11 PM | 22:30 |

Vansh can change these times from the Admin screen.

## Firestore Data Model

```
/tasks_config/{taskId}
  id, title, icon, notificationTime, isDefault, order

/completions/{memberId}_{date}
  memberId, date (yyyy-MM-dd),
  tasks: { taskId: "completed"|"missed" }
```

## Troubleshooting

| Problem | Fix |
|---|---|
| `google-services.json` error | Replace placeholder with real file from Firebase console |
| `firebase_options.dart` has placeholders | Run `flutterfire configure` |
| Notifications not firing | Grant notification permission in Android settings |
| Tasks not syncing | Check Firestore rules are published and internet is on |
| Build fails with NDK error | Run `flutter doctor` and install Android NDK via Android Studio |
